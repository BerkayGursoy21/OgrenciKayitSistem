Projenin amacı Ders kayıt, Öğrenci Kayıt ve Öğretmen kayıt edip, kayıt alınan bilgileri bir csv dosyasına yazdıran java konsol uygulamasıdır.
Ders kayıt bölümünde Ders adı, ders kodu gibi bilgileri alıp ders.csv dosyasına kayıt eder. 

Öğretmen kayıt bölümünde Öğretmenin adı, soyadı, Telefon numarası, Adres ve verdiği dersi seçip alınan bilgileri öğretmen.csv dosyasına kayıt eder.

Öğrenci Kayıt etme bölümüde aynı şekilde çalışmaktadır. Ders seçme işlemi ders.csv dosyasından çekilmektedir.
